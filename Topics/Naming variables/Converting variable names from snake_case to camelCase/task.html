Converting variable names from snake_case to camelCase

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Given a variable name as a text string, your task is to convert it from snake_case style to camelCase style. You will scan a text string as input. The input string will only have alphabetic lower case letters and underscores ('_'). Your output should print the same string but in camelCase style.
Sample Input:
my_variable_name

Sample Output:
myVariableName


Sample Input:
java_coding_problem

Sample Output:
javaCodingProblem
