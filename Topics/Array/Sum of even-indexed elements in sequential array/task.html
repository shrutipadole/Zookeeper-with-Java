Sum of even-indexed elements in sequential array

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Implement a Java program that creates an array of integers with the length specified by user input. The program should populate this array with sequential integers starting from 1. Finally, it should print the sum of the even-indexed elements in the array. The input is a single positive integer n (1 <= n <= 10^6). The output should be a single integer representing the sum of the numbers in the array at the even indices (0, 2, 4, ...).
Sample Input:
3

Sample Output:
4


Sample Input:
5

Sample Output:
9



Memory limit: 256 MB
Time limit: 5 seconds
